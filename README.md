# SU Solutions Landing

Modern, futuristic, production-style IT company landing page built with:

- React
- Vite
- Tailwind CSS

Features:

- Responsive premium landing page sections
- Futuristic glassmorphism + glow visual language
- Light/Dark theme toggle with persisted user preference
- Smooth scrolling, reveal animations, hover transitions

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```
